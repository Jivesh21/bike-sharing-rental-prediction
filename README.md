# 🚴 Bike Sharing Rental Prediction

Machine learning project that predicts bike rental demand using weather and seasonal data.

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis
* Model training
* Rental demand prediction

## Tech Stack

Python, Pandas, NumPy, Scikit-learn, Jupyter

## Files

* `bike_demand.ipynb` → Notebook
* `app.py` → App file
* `model.pkl` → Trained model
